    The purpose of this API  is to provide a way for any GDSC . This api returns 2 json objects,  'past-events' (found in "https://{domain}/api/past-events") and 'upcoming-events' (found in "https://{domain}/api/upcoming-events"). The 'past-events' object contains an array of objects, each object containing the following properties: 'title', 'date', 'time', 'location', 'description', 'link', and 'image'. The 'upcoming-events' object contains an array of objects, each object containing the following properties: 'title', 'date', 'time', 'location', 'description', 'link', and 'image'. The 'image' property is a url to the image of the event. The 'link' property is a url to the event page. The 'date' property is a string of the date of the event. The 'time' property is a string of the time of the event. The 'location' property is a string of the location of the event. The 'description' property is a string of the description of the event. The 'title' property is a string of the title of the event.
